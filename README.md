
## 프로젝트 소개

<p align="justify">
사용자의 league of legend 게임 월간 이용량을 시각화 하는 서비스입니다.
</p>

## 기술 스택

typescript, next.js, tailwindcss, react-query, recoil, shadcn
<br>

## 컨벤션
폴더/파일명 : kebab-case
함수명 : PaskalCase
변수명 : camelCase
